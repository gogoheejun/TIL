# update가 능사가 아니다
update는 데이터가 사라지지 않고 유지되면서 업데이트 될 수 있도록 해준다. Entity모델에 새 칼럼을 추가하면 추가도 해준다.      

그러나, Entity모델에서 칼럼을 삭제를 해도 삭제되지 않음. 그리고 칼럼이름을 바꾸고 싶어도 바뀌지 않고, 바꾼 새로운 칼럼이 생김. 즉 테이블이 지저분해진다. 이럴땐 디비 마이그레이션을 진행해야 함.
